About Project


Our project is about, Prediction using traditional disease risk model usually involves a machine learning and supervised learning algorithm which uses training data with the labels for the training of the models.

Diseases that we are including in our project are common occurring diseases .This increase the importance of our project. Nowadays more and more people are becoming sick so it is necessary to have a software like this which will save the both time and money for the people. The key points are-

⦁ sophisticated, automatic, and objective algorithms for analysis of high-dimensional and multimodal biomedical data.

⦁ improving detection, diagnosis, and therapeutic monitoring of disease.

⦁ Once developed will take any more cost.

⦁ Can be used by both doctors and patients .
